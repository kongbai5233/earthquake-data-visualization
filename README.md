# 地震数据可视化平台（D3.js）

## 📌 项目简介

本项目基于 HTML、CSS、JavaScript 及 D3.js 实现了一个地震数据可视化分析平台。

系统通过对地震数据进行清洗、统计与结构设计，构建多种可视化图表，包括地图分布图、时间趋势折线图及震级统计图，实现了数据的动态交互与多视图联动展示。

---

## 📊 数据来源

- 数据来源：Tableau Public Sample Data
- 数据格式：CSV
- 使用字段：
  - latitude（纬度）
  - longitude（经度）
  - Date（日期）
  - magType（震级类型）
  - place（地理位置）

---

## 🧹 数据处理流程

### 1️⃣ 数据清理

- 删除无关字段
- 提取年份与月份
- 精简地址信息
- 删除缺失值与异常数据

### 2️⃣ 数据统计

- 统计每年地震发生次数（折线图数据）
- 统计不同震级类型数量（矩形图数据）
- 构建每年 12 个月的统计结构
- 转换为适合 D3 渲染的数据结构

---

## 🗺 可视化模块

### 1️⃣ 地图可视化

- 展示地震发生的地理位置
- 使用点标记表示地震
- 通过颜色与大小编码震级信息
- 支持缩放与点击交互
- 悬浮提示信息（d3-tip）

---

### 2️⃣ 年份地震统计折线图

- X 轴表示年份
- Y 轴表示地震发生次数
- 支持点位悬浮提示
- 支持右键跳转至该年份的月份统计图

---

### 3️⃣ 震级分布矩形图

- 展示不同震级类型数量分布
- 面标记面积表示数量大小
- 支持悬浮提示

---

### 4️⃣ 动态年份月份折线图

- 支持：
  - 上一年
  - 下一年
  - 自动播放
- 使用定时器控制动画播放
- 防止多定时器叠加
- 按钮状态动态切换

---

## 🏗 页面结构设计

整体采用左右分栏布局：

- 左侧：功能导航区
- 右侧：可视化展示区

### 技术实现：

- Flex 布局实现全屏响应式
- SVG 作为主要图形渲染方式
- 使用 class 控制视图切换（block / none）
- 背景渐变增强视觉层次
